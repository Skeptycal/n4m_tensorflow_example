Node.js for Max with TensorFlow
====

## What's This

MNIST handwritten character recognitin demo for MAX8.  
Node.js for Max8 + Electron + React + Socket.io + Signature Pad + TensorFlow.js

## How to use

npm run setup  
Open tf_mnist.maxpat
